# Personal Website using a Jeyll Theme Minimal Mistake
## Command

``` bash
  gem install bundler jekyll

  jekyll new my-awesome-site

  cd my-awesome-site

  bundle exec jekyll serve

# => Now browse to http://localhost:4000
```
