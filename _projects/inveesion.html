InVeesion is a computer vision dashboard to analyse and extract information 📈 from an image or video using the most advanced Artificial Intelligence based models.
InVeesion is scalable and accessible across internet thanks to the ☁️ cloud. It is deployed on a secure web server for remote access and usage, thus, you can adjust the computing power and speed to your need.



🔥 Key Features 🔥

  • 🎯 All in one Computer Vision dashboard for image and video processing
  • ⚡ Real-time display of performance and results
  • Save and export results in a .csv file for post analysis in Python / Pandas
  • Record the 🎥 video stream with overlay
  • Deployed in secure web server thanks to the ☁️ cloud technology (Google Cloud, Heroku) for wide accessibility and scalability
  • The user can input an 🖼️ image and 🎞️ video from different sources: URL, upload, demo, IP address
  • Easy to use dashboard with modular architecture for simple personalisation
  • Integrates several AI application such as: Object detection, Fire detection, Object counting, Face detection
  • ... and more
